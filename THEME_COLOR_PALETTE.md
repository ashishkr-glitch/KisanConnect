## ðŸ“± Responsive Behavior

### Light Mode - All Screen Sizes
```
Desktop (>1024px):
  - Full gradient background visible
  - Buttons: #004d66 with smooth hover to #00a699
  - Cards: White with light teal border
  - Tables: Full width with proper spacing

Tablet (768-1024px):
  - Gradient still visible but compressed
  - Buttons: Same color, smaller touch targets
  - Cards: Adjusted padding for space
  - Tables: Horizontal scroll if needed

Mobile (<768px):
  - Gradient visible but vertical flow
  - Buttons: Large touch targets, same color
  - Cards: Full width with margins
  - Tables: Card-style layout or scroll
```

### Dark Mode - All Screen Sizes
```
Same responsive behavior as light mode, but with:
  - Black-to-gray gradient background
  - Off-white text for readability
  - High contrast for small screens
  - Bright status colors visible on all devices
```

## ðŸ”„ Theme Switching Example

### Code to Test Theme Toggle:
```javascript
// In App.js or Theme Switcher component
function toggleDarkMode() {
  const body = document.body;
  if (body.classList.contains('light')) {
    body.classList.remove('light');
    body.classList.add('dark');
    // Colors automatically change to dark mode palette
  } else {
    body.classList.remove('dark');
    body.classList.add('light');
    // Colors automatically change to light mode palette
  }
}
```

## ðŸŽ¨ Color Usage Rules

### Light Mode
- **Use #004d66** for: Buttons, primary actions, important accents
- **Use #00a699** for: Hover states, transitions
- **Use #003d4d** for: Active/pressed states
- **Use #1a1a1a** for: Body text, headings
- **Use #d0e8ed** for: Borders, dividers, subtle separation
- **Use #ffffff** for: Card backgrounds, input fields
- **Use #388e3c** for: Success messages and confirmations
- **Use #f57c00** for: Warning messages and cautions
- **Use #d32f2f** for: Error messages and alerts

### Dark Mode
- **Use #e8e8e8** for: Buttons, primary actions, important accents
- **Use #ffffff** for: Hover states, transitions
- **Use #b0b0b0** for: Active/pressed states
- **Use #e8e8e8** for: Body text, headings
- **Use #333333** for: Borders, dividers, subtle separation
- **Use #1a1a1a** for: Card backgrounds
- **Use #7ddb5f** for: Success messages (bright green)
- **Use #ffb84d** for: Warning messages (bright orange)
- **Use #ff6b6b** for: Error messages (bright red)
